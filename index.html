Геопортал Geolemax - приложение работы с электронными картами, с гео- и атрибутивными данными.
Отечественное программное обеспечение, предназначенное для визуализации, сбора, хранения, обработки географической и атрибутивной информации при помощи браузера.

О системе

О системе

Базовые возможности:
  • Создание и отображение проектов в виде карт
  • Библиотека стилей (создание и редактирования стилей слоев)
  • Таблицы (Создание таблицы с полями различных типов, и типов таблицы один к одном, и один ко многим)
  • Создание векторных надписей над объектами
  • Справочники (простой, каскадный)
  • Настройка поисковой строки
  • Тематический раскараски
  • Дашбоард проекта.
Geolemax позволяет:
  • Выполнять навигацию по картам (увеличение, уменьшение, перемещение);
  • Подключать подложки;
  • Создавать векторные слои;
  • Подключать слои WMS;
  • Работать с данными, импорт и экспорт через shape.
  • При отображении учитывать уровень и вложенность слоев.
  • Поиск и фильтрация объектов.
  • Взаимодействовать с другими программами посредством API.
  • *Создавать/редактировать/удалять объекты.
  • *Интеграция с ГИС Zulu
  • *Интеграцияс ГИС QGis
  • *Интеграция с ГИС Zulu
  • *Настраивается и дорабатывается под конкретные задачи.
Возможности и преимущества:
  • Актуальность векторных и атрибутивных данных
  • Мониторинг транспорта и датчиков автоматизации
  • Простота согласование заявок
  • Статистический анализ
  • Паспортизация объектов
  • Аналитические показатели
  • Единая точка доступа
  • Модульная система
  • Решения задач в разлинчных сферах.

КАК РАБОТАЕТ Geolemax

  • Admin App - веб-приложение для создания/настройки проектов/карт.
  • Other App - веб-приложение для работы конечного пользователя с порталом.
  • API Geolemax - апи, для взаимодействия приложений с бэкэндом.
  • WMS Server - wms сервер, для преобразования векторных данных в картинку.
  • Tile Cache - кэширование wms слоев, для увеличения призводительности.
  • DB - база данных.

МОДУЛИ И НАПРАВЛЕНИЯ

Geolemax - имеет возможность объединить в себе различные области, организации в одно месте. Возможно вывести важные показатели на объекты, арезультаты на дашбоард.
